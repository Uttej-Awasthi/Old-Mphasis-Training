Component Functionality

Child Records Display:

When a user is viewing a specific record (e.g., an Account), the component will display related child records (e.g., Opportunities) in a data table.
The fields to display for these child records can be configured using fieldsets in the Lightning App Builder.
Grandchild Records Display:

Each child record can be expanded (using a chevron icon) to display related grandchild records (e.g., OpportunityLineItems).
The fields to display for these grandchild records are also configurable using fieldsets.
Dynamic Configuration:

The component is designed to be generic and configurable for any parent-child-grandchild object relationship.
Admins can set the child and grandchild object API names and their corresponding fieldsets in the Lightning App Builder, allowing the component to work with different objects without code changes.
